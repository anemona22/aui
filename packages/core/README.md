# INSTALLATION

Install the plugin **using npm**

```sh
npm install @ideoui/core --dev
```

**or using yarn**

```sh
yarn add @ideoui/core --dev
```

Then add the plugin to your `tailwind.config.js` file

```js
// tailwind.config.js
module.exports = {
  ...
  plugins: [
    require(@ideoui/core)
  ]
}
```
